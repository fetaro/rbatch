RBatch::CommonConfig

Read common config file and return hash opject.

Default common config file path is “../conf/common.yaml“

Sample

config : ./conf/common.yaml 

 key: value
 array:
  * item1
  * item2
  * item3

script : ./bin/sample.rb 

 require 'rbatch'
 p RBatch::common_config
 => {"key" => "value", "array" => ["item1", "item2", "item3"]}
